Introducing SampleSite

SampleSite is not a real company or entity. This page was developed to present valid and semantic XHTML. All hyperlinks on this page are disabled and the CSS styles reside in an external stylesheet which can be discovered by viewing the source code.

This page uses XHTML 1.0 strict but it has been coded using class names that mimic the new semantics introduced in HTML5. Introducing HTML5 semantics within an XHTML page makes for an easier transition to future document standards. The class names of importance in this document as they relate to HTML5 include header, nav, aside, section and footer.

This page passed validation successfully at http://validator.w3.org
