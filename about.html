Webfolio is a web portfolio for open source front end development. The website serves as a live code sample where the focus is on the clean,semantic and standards compliant HTML5 and CSS with a touch of Javascript,jQuery,PHP and MySQL.

By complementing HTML5 and CSS with a server-side language such as PHP and a database system such as MySQL it is possible to create data-driven websites.

PHP can be used to communicate with MySQL to retrieve data and then embeds the data into an existing HTML page as if it was originally written in the source code. Since the data from the database is embedded in the existing HTML page the page structure and styling remain the same while the data retrieved from the database can change and is reflected in the webpage.
