Webfolio projects consist of short code samples that illustrate various open source web development techniques and technologies. As there are many open source technologies Webfolio attempt to limit the projects to HTML5, CSS, Javascript, jQuery, PHP and MySQL.

Some projects are meant to illustrate proper coding. Other projects are meant to be interactive and require the user to click on buttons, move the mouse, enter input using the keyboard and so on. Look at the page contents to decifer if you should be looking at the source code or interacting with the page.
