Webfolio is a live code sample geared towards presenting open source front end web development technologies including but not limited to HTML5, CSS, Javascript, jQuery, PHP and MySQL.

Emphasis has been placed on writing clean and semantic HTML5 and CSS source code which adheres to W3C standards.

Feel free to click through the links, view the source code or validate the pages with the World Wide Web Consortium's validator which can be found at http://validator.w3.org/

See the sidebar for links to external sites that provide tools, descriptions and official documentation.
